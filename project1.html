About the Project

Project Name: Ecommerce Website For True Stones

Skills Used: HTML, CSS, PHP, JavaSscript, Mysql Database

About: The main aim of this project was to develop an Ecommerece website for stones which we normally use for designing of houses, offices etc. This project was mainly designed with all the necessary features that an Ecommerce site is needed starting from product description, Selecting product, Logins for user and Admin and payment Processing etc..

Successfully developed user friendly website with easy navigation for users throughout the website.

Successfully designed checkout page and payment processing for the selected products using paypal API.

Created a review system for each product to ensure that the particular product is best to purchase or not.

Developed a backend management system that allows authorized person(admin) to make updates to the site and manage user accounts, reviews, adding new products etc.
