About the Project

Project Name: Simple Notes App using React

Skills Used: HTML, CSS, JavaScript, React, Fetch API(Axios)

About: The main aim of this project was to develop a Simple Notes App where user can write notes, edit and delete the notes.

Developed a Simple CRUD based notes app using react and Fetch API using Axios where user can post the notes, edit the notes and delete the notes and search for specific notes.

Used React JS concepts like useState, useEffect, React Router etc for developing the app.
