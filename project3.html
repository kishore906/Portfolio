About the Project

Project Name: Information Technology Careers and College Search SystemProject

Skills Used: HTML, CSS, JavaScript, Node JS, Chart JS, Mysql Database

About: The main aim of this project was to develop a project for students about different careers in Information Technology field like DataScience, Web Development, Cloud Computing etc.

This Project mainly provides information on various IT specializations like, web development, Data Science and Cloud Computing along with programming languages to learn and some useful resources links and a chart showing salaries in different countries for each role in a particular career.

User Authentication system was built using Node JS and a search system was developed to search for colleges in different states (Only Australia) depending on the selected degree.

Users can view the details of the university along with university details like fee, total units to complete during the course, rating etc along with near accomodations. And users can create a list of their interested universities for future view.
