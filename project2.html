About the Project

Project Name: Blog Posts(CMS) Project

Skills Used: HTML, CSS, PHP, JavaScript, Mysql Database and BootStrap Framework

About: The main aim of this project was to develop a Content Management System with an idea of displaying posts depending upon the categories selected. The project was designed with all necessary features starting from login system for users and admin, posts displaying, giving reviews to particular post etc..

Designed user interactive web pages using latest web technologies and bootstrap framework.

Successfully implemented password hashing technique for the users and sessions were created for particular user login.

Authenticated users can post, edit, delete their post and can give reviews to other posts.

Created a dashboard for the project in the backend to manage the content of the website and user’s data, reviews approval and editing the content, etc.

Used charts to display data from database in a graphical view.
